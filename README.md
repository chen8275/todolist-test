# 待办事项 Todo List

一个基于 React + TypeScript + Tailwind CSS 构建的现代化待办事项应用。

**在线访问：** https://chen8275.github.io/todolist-test/

## 功能

- 添加、编辑、删除任务
- 标记任务完成状态
- 按状态筛选（全部 / 未完成 / 已完成）
- 一键清除已完成任务
- 数据本地持久化（localStorage）
- 完成任务时撒花动效

## 技术栈

- React 19
- TypeScript
- Vite 6
- Tailwind CSS 4
- Lucide React（图标）
- Motion（动画）
- Canvas Confetti（撒花效果）

## 本地开发

```bash
npm install
npm run dev
```
